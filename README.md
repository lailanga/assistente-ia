# Assistente IA

Assistente pessoal via WhatsApp utilizando:

- n8n
- OpenAI
- PostgreSQL
- Meta WhatsApp Cloud API

## Objetivo

Criar um assistente capaz de:

- Criar tarefas
- Atualizar tarefas
- Consultar tarefas
- Conversar naturalmente
- Manter contexto
- Evoluir para agenda e memória pessoal

## Tecnologias

- n8n
- PostgreSQL
- OpenAI GPT
- Meta Cloud API
- Docker
